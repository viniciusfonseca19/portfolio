02 — Sobre Mim

Código limpo,
soluções reais.

Sou um desenvolvedor fullstack apaixonado por criar sistemas que funcionam de verdade. Com sólida base em Java e Spring no backend, e domínio de React no frontend, construo aplicações completas — do banco de dados à interface.

Acredito que bom software é aquele que resolve problemas com elegância: código bem arquitetado, APIs bem documentadas e UIs que fazem sentido para quem usa. Boas práticas não são opcionais — são o padrão.
